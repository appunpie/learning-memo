## わかったこと  
<ins>dequeについて</ins>  
```python
from collections import deque
q = deque()

q.append(1)
q.append(2)
q.appendleft(0)
print(q)
>>>deque([0, 1, 2])
```  
リストではリスト内の要素を追加・削除するには最後尾の要素にしか操作を行うことができなかった。  
dequeでは、先頭からの要素の追加・削除ができるため、かなり使い勝手がいいと言える。

<ins>joinについて</ins>  
```python
ans = [1, 2, 3, 4]
s = ' '.join(map(str, ans))
print(s)
>>>>1 2 3 4
```  
joinを用いることでリスト内の要素に空白を付けて出力が可能  
※ただし、intをstrの型に変えないとつかえないため、map関数を使っている
